# 镇江香醋风味评价标准 v2.0

> **基于 GB/T 18623-2011《地理标志产品 镇江香醋》优化**
> 制定日期: 2026-06-28
> 修订日期: 2026-06-28 (v2.0)
> 数据基础: 国标 + 4 篇核心文献 + 9 种有机酸风味阈值
> 用途: 自动分级 + 国标符合性检验 + 模型评价

---

## 📋 引用国标

**GB/T 18623-2011** 地理标志产品 镇江香醋 (含第1/2/3号修改单)

主要理化指标 (强制性):
| 指标 | 单位 | 优级品 | 合格品 | 检验方法 |
|---|---|---|---|---|
| 总酸 (以乙酸计) | g/100mL | **≥ 4.5** | **≥ 3.5** | GB/T 5009.41 |
| 不挥发酸 (以乳酸计) | g/100mL | ≥ 0.5 | ≥ 0.5 | GB/T 5009.41 |
| 还原糖 (以葡萄糖计) | g/100mL | ≥ 1.0 | ≥ 1.0 | GB/T 5009.7 |
| 氨基酸态氮 | g/100mL | ≥ 0.18 | ≥ 0.18 | GB/T 5009.39 |
| 食盐 (以NaCl计) | g/100mL | ≥ 0.5 | - | GB/T 5009.44 |

**GB/T 18187-2000** 酿造食醋 (通用):
- 总酸 ≥ 3.5 g/100mL

## 🏆 等级体系 v2.0 (符合国标)

### 强制指标 (国标硬性要求)
- 总酸 ≥ 3.5 (合格) / ≥ 4.5 (优级)
- 不挥发酸 ≥ 0.5
- 还原糖 ≥ 1.0
- 氨基酸态氮 ≥ 0.18

### 风味指标 (基于文献的扩展评价)
- 柔和度 (不挥发酸占比)
- 鲜度 (Glu+Asp 浓度)
- 酯香 (乙酸乙酯+异戊酯 OAV)
- 川芎嗪 (TTMP 浓度)
- 感官等级 (文献中实测的感官评分)

### 综合等级 (v2.0)
| 等级 | 总分阈值 | 国标级别 | 风味特征 |
|---|---|---|---|
| ★★★★★ **特级陈酿** | ≥ 90 | 优级+ | 8年陈, 川芎嗪丰富 |
| ★★★★ **优级** | 75 - 89 | 优级 | 3-5年陈酿, 风味协调 |
| ★★★ **合格** | 60 - 74 | 合格 | 1-3年, 基本达标 |
| ★★ **普通** | 45 - 59 | 合格- | 新醋, 需陈酿 |
| ★ **次品** | < 45 | 不合格 | 多项不达标 |

### 国标符合性判定
- **国标优级**: 4 项强制指标全部 ≥ 优级阈值
- **国标合格**: 4 项强制指标全部 ≥ 合格阈值
- **不合格**: 任一项 < 合格阈值

## 🎯 综合评分 (v2.0)

```
总分 = 酸度分(25) + 不挥发酸分(15) + 还原糖分(10) 
     + 氨基酸态氮分(15) + 柔和度分(10) + 酯香分(15) + 川芎嗪分(10)
满分 = 100
```

| 维度 | 上限 | 评分依据 |
|---|---|---|
| 酸度 | 25 | 国标总酸分级 |
| 不挥发酸 | 15 | 国标不挥发酸分级 |
| 还原糖 | 10 | 国标还原糖分级 |
| 氨基酸态氮 | 15 | 国标氨基酸态氮分级 |
| 柔和度 | 10 | 不挥发酸占比 |
| 酯香 | 15 | OAV 加和 |
| 川芎嗪 | 10 | TTMP 浓度 |

## 🧪 应用示例

```python
from standard import classify_sample, check_gb_compliance

sample = {
    "总酸": 7.20, "不挥发酸": 2.21, "还原糖": 2.64,
    "氨基酸态氮": 0.21, "谷氨酸": 300, "天冬氨酸": 27,
    "乙酸乙酯": 2699, "乙酸异戊酯": 80, "四甲基吡嗪": 30,
}

# 风味分级
result = classify_sample(sample)
# → {总分: 90, 综合等级: "★★★★★ 特级陈酿"}

# 国标符合性
gb = check_gb_compliance(sample)
# → {总酸: 优级(7.20≥4.5), 不挥发酸: 优级(2.21≥0.5),
#    还原糖: 优级(2.64≥1.0), 氨基酸态氮: 优级(0.21≥0.18),
#    4项_全合规: True}
```
