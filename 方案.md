研究加工过程参数与特征风味物质之间关联模型
一、项目概况（研究背景与意义）
传统食醋（以镇江香醋为代表）风味高度依赖固态发酵、淋醋、陈酿等复杂、长周期、多变量耦合工艺，风味物质（有机酸、酯类、醇类、醛酮、氨基酸、酚类等）由微生物代谢、酶促反应、非酶褐变共同决定。
当前痛点集中于：工艺参数（温度、水分、盐度、酸度、时间、翻醅频次、氧气、接种量、原料配比）与风味物质关联性模糊、经验化、不可量化；缺乏定量关联模型，无法精准预测风味、指导参数优化、实现稳定化生产；风味波动大、批次一致性差、老技师经验难以传承；智能化升级缺乏数据驱动的核心模型底座。
本研究以镇江香醋典型生产工序（酒精发酵、醋酸发酵、淋醋、陈酿）为对象，构建工艺参数—特征风味物质的多维关联模型，实现：参数可量化、风味可预测、过程可调控、品质可稳定、经验可传承，为传统调味品智能加工提供核心算法支撑，助力恒顺醋业新质生产力转型。
二、主要研究内容（四大模块）
模块 1：全工序关键工艺参数与风味物质系统筛选
1.1 工艺参数筛选（可控变量）
覆盖香醋全流程，筛选15-20个核心可控参数：
原料：糯米淀粉含量、水分、粉碎度、酒曲添加量、
酒精发酵：初始糖度、温度（28-35℃）、发酵时间、pH、酵母菌接种量
醋酸发酵：醅温（32-45℃）、初始酸度、水分、盐含量、翻醅频次、通风量、发酵周期、醋酸菌接种量
陈酿：温度、湿度、陈酿时间、容器材质、氧气接触量
1.2 特征风味物质靶向筛选（响应变量）
查阅文献，锁定25-30个核心风味物质（覆盖香、酸、鲜、醇、焦香）：
有机酸：乙酸、乳酸、琥珀酸、柠檬酸、苹果酸（决定酸度与柔和度）
酯类：乙酸乙酯、乙酸异戊酯、乳酸乙酯、己酸乙酯（果香 / 酯香）
醇类：乙醇、正己醇、异戊醇（醇香）
醛酮：乙醛、糠醛、5 -羟甲基糠醛（焦香/熏香）
氨基酸：谷氨酸、天冬氨酸、丙氨酸（鲜味）
酚类：愈创木酚、4 -乙基愈创木酚（酱香 / 烟熏香）
1.3 数据采集方案
实验设计：查阅文献，覆盖参数高低水平，构建多维数据矩阵；
采样节点：每工序关键时间点（发酵第1/3/5/7 天、陈酿 1/3/6/12个月）；
模块 2：多维数据融合与风味指纹图谱构建
2.1 多源数据预处理
工艺参数：归一化、标准化、异常值剔除、缺失值插补；
风味物质：峰面积校正、定性定量、相对含量计算；
感官数据：量化评分标准化、一致性检验（克朗巴哈系数）。
2.2 风味指纹图谱建立
采用多元统计分析（PCA、PLS-DA、聚类分析），筛选关键差异风味标志物，构建：
工序特异性指纹（酒精发酵 / 醋酸发酵 / 熏醅 / 陈酿）；
品质等级指纹（优 / 良 / 合格）；
批次稳定性指纹。
实现 “一组数据 = 一个风味特征 = 一套品质等级”，将主观感官转化为客观数字指纹。
模块 3：工艺参数—风味物质关联模型构建（核心创新）
3.1 基础关联分析
相关性分析：Pearson/Spearman 相关系数，筛选强关联参数 - 物质对；
回归模型：多元线性回归（MLR）、逐步回归、岭回归、LASSO 回归，建立初步定量方程；
典型相关分析（CCA）：揭示参数组与风味组的整体耦合关系。
3.2 非线性与动态关联模型（重点突破）
传统线性模型难以刻画发酵体系强非线性、时变性、耦合性，本研究采用机器学习 + 深度学习融合建模：
随机森林（RF）/ 梯度提升树（XGBoost/LightGBM）：挖掘参数间交互作用，实现风味物质精准预测，输出参数重要性排序；
支持向量机（SVR）：小样本、高维数据下的非线性回归，适合发酵过程小批量实验；
长短期记忆网络（LSTM）：捕捉时序动态特征，模拟发酵过程风味物质随时间的演变规律，实现动态预测；
混合模型（ML+DL）：前期工艺参数用 XGBoost，时序发酵过程用 LSTM，融合输出高精度预测结果。
3.3 模型验证与优化
数据集划分：70% 训练、15% 验证、15% 测试；
评价指标：R²（决定系数）、RMSE（均方根误差）、MAE（平均绝对误差）、MAPE（平均绝对百分比误差）；
模型优化：网格搜索、交叉验证、超参数调优，确保模型稳定性、泛化性、鲁棒性；
显著性检验：方差分析（ANOVA）、t 检验，验证参数对风味物质影响的统计学意义。
模块 4：模型应用与智能加工决策系统开发
4.1 风味预测与参数优化
输入：工艺参数组合；
输出：各特征风味物质含量、感官评分、品质等级；
优化：基于模型反向推导最优工艺参数区间，实现 “目标风味→参数推荐”。
4.2 智能反馈控制模型
构建实时数据采集 — 模型预测 — 参数微调闭环系统：
在线传感器：温度、pH、溶氧、酸度实时采集；
边缘计算：模型部署于车间终端，实时预测风味；
自动调控：联动发酵罐温控、翻醅机、通风系统，自动修正参数偏差。
4.3 成果验证与示范
恒顺醋业中试线放大试验，验证稳定性；
生产示范：1–2 条生产线试点应用，对比传统工艺：风味一致性提升≥90%、批次差异降低≥50%、优质品率提升≥15%。
